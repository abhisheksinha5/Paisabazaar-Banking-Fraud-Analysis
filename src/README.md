# Source Code

This folder is reserved for reusable Python scripts and modules used in the project.

The current version of this project is primarily notebook-based and does not contain standalone source files. Future improvements may include:

- `data_cleaning.py` – Data cleaning and preprocessing functions
- `eda.py` – Exploratory Data Analysis functions
- `visualization.py` – Plotting and visualization utilities
- `utils.py` – Helper functions and reusable code
